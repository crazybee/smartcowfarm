SmartCowFarm is an Azure-native livestock monitoring solution that combines IoT telemetry, serverless processing, real-time alerts, and a web dashboard to help farmers track cow health and operations. Sensor data is ingested through Azure Event Hub/IoT-style messaging, processed by Azure Functions, and stored in SQL Server via Entity Framework. The backend exposes REST APIs for cow records, vaccinations, and alerts, while scheduled timer functions proactively flag vaccinations due in the next few days. A geofencing and health rules engine generates alerts for high temperature, boundary breaches, and vaccination risk, and pushes them live to users through Azure SignalR. On the frontend, a Vue 3 application provides dashboard analytics, map-based herd tracking, cow management, and vaccination workflows, giving farm operators a single control center for day-to-day monitoring and early intervention.
